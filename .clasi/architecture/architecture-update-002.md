# Architecture Update -- Sprint 002: README Tagline

## What Changed

No architectural changes. This sprint makes a single documentation edit:
a one-sentence tagline is added to README.md below the top-level heading.

No modules, components, interfaces, data models, or dependencies are added,
modified, or removed.

The issue `update-readme-tagline` identified that the README lacked an
immediate description of the system's purpose. The change improves first
impression for contributors and stakeholders browsing the repository (SUC-001).

None. README.md is a static documentation file. It has no runtime role and
no references from source code or build tooling.

## Migration Concerns

None.
