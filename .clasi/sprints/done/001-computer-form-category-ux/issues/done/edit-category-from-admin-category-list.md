# Edit category from admin category list

## Description

In the category list on the administration pages, the category name
should be clickable. Clicking it opens an editor where the user can
modify the category (rename it, change any other editable fields).

### Scope

- Make the category name in the admin category list a clickable
  link / button.
- Clicking it navigates to (or opens) an edit form for that category.
- The form supports editing the category's editable fields and
  saving changes back.
