# Sprint 034 Use Cases

## SUC-001: Compact Issues List with Search
Parent: UC-3.3

- **Actor**: Quartermaster
- **Preconditions**: Issues exist in the system
- **Main Flow**:
  1. User views the issues list
  2. Issues are displayed in compact two-line rows
  3. User types in the search box to filter issues
- **Postconditions**: Filtered issues displayed compactly
- **Acceptance Criteria**:
  - [ ] Issues display in compact two-line format
  - [ ] Search input filters issues by text

## SUC-002: Kit List Last Updated Column
Parent: UC-5.1

- **Actor**: Quartermaster
- **Preconditions**: Kit list displayed
- **Main Flow**:
  1. User views kit list
  2. Sees Last Updated column with relative dates
  3. Clicks to sort by most recently updated
- **Postconditions**: Kit list sorted by last updated
- **Acceptance Criteria**:
  - [ ] Last Updated column appears on kit list
  - [ ] Column is sortable
