# Pack list date columns

## Description

The pack list should include "Created" and "Last Updated" columns so
users can see when packs were added and last modified.

## Acceptance Criteria

- [ ] Pack list table includes "Created" column
- [ ] Pack list table includes "Last Updated" column
- [ ] Dates formatted readably (relative time or short date)
