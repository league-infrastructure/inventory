# Dashboard: Replace "Transferred Out" with Recent Custody Changes

The dashboard currently has a "Transferred Out" section which isn't
useful. Replace it with a **Recent Custody Changes** section that shows
the full context of each transfer:

- Who transferred it
- To whom it was transferred
- At which site
- When it happened

Example: "Kit #42 transferred from Jane Doe to John Smith at Downtown
Site — 2 hours ago"

This gives a meaningful activity feed rather than just a list of items
that left.
