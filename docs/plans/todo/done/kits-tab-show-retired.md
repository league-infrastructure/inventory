# Kits tab should show retired kits

## Summary

The Kits tab needs a way to view retired kits so users can find and
un-retire them. Currently, once a kit is retired it effectively
disappears from the UI with no way to bring it back.

## Expected Behavior

- The Kits tab should have a filter or sub-section for retired kits
- Each retired kit should have an action to restore it to ACTIVE status
